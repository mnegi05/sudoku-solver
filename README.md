# Sudoku Solver

Cross-platform Expo app for solving Sudoku puzzles on mobile and web.

## Features

- Choose between `6 x 6` and `9 x 9` Sudoku boards.
- Enter puzzle values in an empty grid.
- Solve the puzzle on the same screen.
- Show user-entered values in bold black and solved values in green.

## Run

```bash
npm install
npm run web
```

Use `npm run android` or `npm run ios` for mobile.
